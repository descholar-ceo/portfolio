Stay in touch

Stay in touch is a web application that helps people to get connected with whoever no matter how far away they are.

I have done it full-stack, means that I developed it from database design to the frontend design

Note: The result of this project is only an MVP: Minimal Viable Product. but it is very functional, some of the features we see in other Stay in touchs such as facebook and twitter, for example notifications system aren't yet implemented but they will be in the very final product.
